# HyperOS 3 Lite Blur Control Center (No Animation)

A lightweight systemless module that enables the **Lite Blur Control Center** on **HyperOS 3 (Android 15)** devices without applying Control Center opening or closing animations. Fully compatible with both **Magisk** and **KernelSU**, this module is designed for users who prefer a clean, responsive, and minimal Control Center experience while maintaining maximum system performance and stability.

Unlike flagship devices that use resource-intensive real-time blur effects, this module implements a lightweight **Lite Blur** solution that enhances the appearance of the Control Center without increasing GPU usage. The **No Animation** edition preserves the default animation behavior of the system, making it an excellent choice for users who prefer a faster and simpler user experience.

---

## Features

- ✨ Enables Lite Blur Control Center
- 🚫 No Control Center opening or closing animations
- ⚡ Lightweight and performance-friendly
- 🔒 Fully compatible with both **Magisk** and **KernelSU**
- 📱 Designed exclusively for **HyperOS 3 (Android 15)**
- ♻️ Systemless installation with easy removal
- 🚀 Premium Control Center appearance without real-time blur
- 💯 Stable, optimized, and suitable for daily use

---

## Requirements

- HyperOS 3
- Android 15
- Root access
- **Magisk** or **KernelSU**

> **Compatibility:** This module is designed **only for HyperOS 3 running Android 15**. It is **not compatible** with HyperOS 1, HyperOS 2, or other Android versions.

---

## Installation

### Magisk

1. Download the latest release.
2. Open the **Magisk** application.
3. Navigate to **Modules**.
4. Tap **Install from Storage**.
5. Select the downloaded ZIP file.
6. Reboot your device.

### KernelSU

1. Download the latest release.
2. Open the **KernelSU Manager** application.
3. Navigate to **Modules**.
4. Install the downloaded ZIP file.
5. Reboot your device.
